<p>在这篇文章中，我将分享如何使用 Next.js 和 shadcn/ui 组件库构建一个现代化的个人博客网站。</p>
<h2 id="-nextjs-0">为什么选择 Next.js？</h2>
<p>Next.js 是一个基于 React 的框架，它提供了许多开箱即用的功能，如：</p>
<ul>
<li><strong>服务器端渲染 (SSR)</strong> - 提高首屏加载速度和 SEO</li>
<li><strong>静态站点生成 (SSG)</strong> - 预渲染页面，提供极快的加载体验</li>
<li><strong>增量静态再生成 (ISR)</strong> - 结合了 SSG 和 SSR 的优点</li>
<li><strong>文件系统路由</strong> - 简化路由配置</li>
<li><strong>API 路由</strong> - 轻松创建 API 端点</li>
</ul>
<h2 id="shadcnui--1">shadcn/ui 组件库简介</h2>
<p>shadcn/ui 不是传统意义上的组件库，而是一套可复制粘贴的组件集合，它基于 Tailwind CSS 和 Radix UI 构建。使用 shadcn/ui 的主要优势包括：</p>
<ol>
<li><strong>完全可定制</strong> - 组件代码完全由你掌控</li>
<li><strong>无依赖性</strong> - 不是一个需要安装的包，没有版本冲突</li>
<li><strong>优秀的可访问性</strong> - 基于 Radix UI 原语</li>
<li><strong>美观的设计</strong> - 现代且优雅的默认样式</li>
</ol>
<h2 id="-2">项目设置</h2>
<p>首先，我们需要创建一个新的 Next.js 项目：</p>
<pre><code class="language-bash">npx create-next-app my-blog --typescript --tailwind --eslint
</code></pre>
<p>然后，安装 shadcn/ui CLI 工具：</p>
<pre><code class="language-bash">npx shadcn-ui@latest init
</code></pre>

<li>添加标签过滤功能</li>
</ol>
