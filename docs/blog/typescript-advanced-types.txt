2e:Te1e,<html><head></head><body><h1>TypeScript 高级类型技巧</h1>
<p>TypeScript 的类型系统非常强大，掌握高级类型可以让你的代码更加健壮和灵活。本文将深入探讨一些 TypeScript 中的高级类型技巧。</p>
<h2 id="-0">条件类型</h2>
<p>条件类型允许我们基于类型关系表达式来选择类型：</p>
<pre><code class="language-typescript">type IsString&lt;T&gt; = T extends string ? true : false;

// 使用示例
type A = IsString&lt;string&gt;; // true
type B = IsString&lt;number&gt;; // false
</code></pre>
<h2 id="-1">映射类型</h2>
<p>映射类型允许我们从现有类型创建新类型：</p>
<pre><code class="language-typescript">type Readonly&lt;T&gt; = {
  readonly [P in keyof T]: T[P];
};

// 使用示例
interface Person {
  name: string;
  age: number;
}

type ReadonlyPerson = Readonly&lt;Person&gt;;
// 等同于：
// {
//   readonly name: string;
//   readonly age: number;
// }
</code></pre>
<h2 id="-infer--2">类型推断与 infer 关键字</h2>
<p><code>infer</code> 关键字允许我们在条件类型中推断类型：</p>
<pre><code class="language-typescript">type ReturnType&lt;T&gt; = T extends (...args: any[]) =&gt; infer R ? R : any;

// 使用示例
function add(a: number, b: number): number {
  return a + b;
}

type AddReturnType = ReturnType&lt;typeof add&gt;; // number
</code></pre>
<h2 id="-3">联合类型与交叉类型</h2>
<p>联合类型表示一个值可以是几种类型之一：</p>
<pre><code class="language-typescript">type StringOrNumber = string | number;
</code></pre>
<p>交叉类型表示一个值同时满足多种类型：</p>
<pre><code class="language-typescript">type PersonWithSkills = Person &amp; { skills: string[] };
</code></pre>
<h2 id="-4">实用工具类型</h2>
<p>TypeScript 内置了许多实用的工具类型：</p>
<ul>
<li><code>Partial&lt;T&gt;</code>: 将所有属性设为可选</li>
<li><code>Required&lt;T&gt;</code>: 将所有属性设为必需</li>
<li><code>Pick&lt;T, K&gt;</code>: 从 T 中选择特定属性</li>
<li><code>Omit&lt;T, K&gt;</code>: 从 T 中排除特定属性</li>
<li><code>Record&lt;K, T&gt;</code>: 创建键类型为 K，值类型为 T 的对象类型</li>
</ul>
<h2 id="-5">高级示例：类型安全的事件发射器</h2>
<p>下面是一个使用高级类型实现的类型安全事件发射器：</p>
<pre><code class="language-typescript">type EventMap = {
  click: { x: number; y: number };
  focus: undefined;
  input: { value: string };
};

class EventEmitter&lt;T extends Record&lt;string, any&gt;&gt; {
  private listeners: { [K in keyof T]?: ((data: T[K]) =&gt; void)[] } = {};

  on&lt;K extends keyof T&gt;(event: K, listener: (data: T[K]) =&gt; void): void {
    if (!this.listeners[event]) {
      this.listeners[event] = [];
    }
    this.listeners[event]?.push(listener);
  }

  emit&lt;K extends keyof T&gt;(event: K, data: T[K]): void {
    this.listeners[event]?.forEach(listener =&gt; listener(data));
  }
}

// 使用示例

// 类型安全的事件监听

emitter.on('input', data =&gt; console.log(data.value));

// 类型安全的事件发射
emitter.emit('click', { x: 10, y: 20 });
emitter.emit('input', { value: 'hello' });
</code></pre>
